Witchy Design System

Typography

WDS's typography is mainly based on the fonts Inter and STIX Two with the latter being used mostly for content headings (i.e. ones inside an <article> element)

Basic typographical styles

You can use the <mark> tag to highlight text.

Text inside <del> is meant to be treated as deleted text.

Text inside <s> is meant to be treated as no longer accurate.

Text inside <ins> is meant to be treated as an addition to the document.

Text inside <u> will render as underlined.

Text inside <small> is meant to be treated as fine print.

Text inside <strong> is rendered as bold text.

Text inside <em> is rendered as italicized text.

Text with .text-muted is rendered as muted text.

Multilingual typography

WDS supports RTL text like اللغة العربية and השפה הערבית as well as ruby text like のんびり (nonbiri).
